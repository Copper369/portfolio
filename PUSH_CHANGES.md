# Push Changes to GitHub

Run these commands in your terminal:

```bash
git add .
git commit -m "Update portfolio: add mac-style cards, remove contact form, 2-column mobile certs"
git push portfolio main
```

## Changes in this commit:
- Added macOS-style window controls to hobby project cards
- Removed contact form, kept only Say Hello section
- Updated certifications to show 2 columns on mobile view
- Improved mobile responsiveness for certificates

## After pushing:
Netlify will automatically detect the changes and deploy your site.
Check your Netlify dashboard to monitor the deployment.
